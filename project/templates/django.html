Welcome to the Django Section


After learning basics of Python, I looked at areas I could apply the skills I had learnt to real world projects. I followed a tutorial to produce a Task List Web app. This introduced me to Django and using MVC frameworks. Through this course I learnt the basics about urls, views, models and touched on using HTML to create the pages. I have continued to develop my skills by adding further features to this app.

Please click to see web page Link to Web App


I have added features to the site to change the use to a recruitment tool. The concept of the site is to allow shift workers to update their availability, this is then automatically matched with available shifts needing their skill set. The first section I added calendar section where people can mark which hours they are free to work.


There is a simple profile which has user updateable information such as email, username. It also holds information that the user cannot update this is used to hold information such as DBS number and which areaâ€™s they can work.



The finial section is an area for shifts to be added, this is area the user can add what kind of worker they require and the times they are needed. Once a shift is added the web app checks whether there is anyone has marked themselves available to work and automatically books them to the shift.

If they are booked onto the shifts it updates into their calendar to show the shift they are working.


What do I think of the site?

2. Use Docker for my development and production environments
3. Use custom user model
4. TESTS! Sadly when I started I did not recognise the importance of this.

So I am currently rebuilding the site from the ground up to do it again implementing the new best practices I have learnt since I first built the app.
